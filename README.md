# Customer Management System (Spring Boot)

A simple backend project built with **Spring Boot** that provides basic **CRUD operations** for managing customers.  
This project includes APIs for:

- Saving a new customer  
- Retrieving all customers  
- Finding a customer by ID  

It is ideal for beginners who want to learn how to build REST APIs using Spring Boot and Spring Data JPA.
